git version - отобразится версия git
git config--global user.name "ник" - для установки имени и отправки проектов на github
git config--global user.email "почта@yandex.ru" - для установки почты и отправки проектов на github
cd - для перехода в папки
mkdir - для создания папки с каким либо названием
git init - создаёт новый локальный Git-репозиторий
git remote add origin ссылка_на_репозиторий - привязка удалённого репозитория к локальному
git status - проверка состояния репозитория 
git add . - добавление файлов в индекс

Для работы с GitHub:
git push - для отправки на GitHub
git commit - для создания коммитов
git add . - добавление файлов в индекс
